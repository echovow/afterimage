\# AFTERIMAGE (v0)



A web arcade prototype where your last seconds return as looping ghosts.



\## Run locally

Just open `public/index.html` or serve the folder:



\- `python -m http.server` (from repo root) then open `/public/`



\## Deploy (Cloudflare Pages)

\- Build command: none

\- Output directory: `public`



\## Controls

\- WASD / Arrow keys: move

\- R: restart

\- Space: pause
